# Задача Эйнштейна

### Запуск приложения
1. Клонируйте проект из репозитория.

**Клиент**

2. Установка зависимостей
```
cd ide-client
npm install
```
3. Запуск клиента
```
npm start
```

**Сервер**

4. Установка зависимостей
```
cd ..
cd ide-server
pip3 install -r requirements.txt
```
5. Запуск сервера
```
. venv/bin/activate
flask run
```
